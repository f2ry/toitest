Алфавит

Алфавитом называют набор знаков, в котором установлен порядок их следования. Разделяют первичный и вторичный алфавиты.

Первичный алфавит

Первичным алфавитом называют алфавит, с помощью которого информация представляется до преобразования.

Вторичный алфавит

Вторичным алфавитом называют алфавит, с помощью которого информация представляется после преобразования.

Код

Код есть правило, описывающие соответствие знаков или их сочетаний одного алфавита знаками или их сочетаниями другого алфавита.

Кодирование

Кодирование есть перевод информации, представленной посредством первичного алфавита, в последовательность кодов. Разделяют алфавитное, блочное, и двоичное кодирование.

Алфавитное кодирование

При алфавитном кодировании коды строятся для каждого знака первичного алфавита.

Блочное кодирование

При блочном кодировании коды строятся для комбинаций знаков первичного алфавита.

Двоичное кодирование

Под двоичным кодированием понимается метод представления информации с использованием двух символов, обычно 0 и 1, представляя каждый исходный символ уникальной последовательностью битов.

Декодирование

Декодирование есть процесс, обратный кодированию, или же восстановление информации в первичном алфавите по полученной последовательности кодов.

Равномерный код

Равномерным кодом называется такой код, который формирует для всех символов первичного алфавита комбинации одинаковой длины.

Неравномерный код

Неравномерным кодом называется такой код, который формирует кодовые комбинации разной длины.

Оптимальный код

Оптимальным кодом называется такой код, избыточность которого минимальна для источника с данными характеристиками.

Избыточность

Избыточность есть превышение количества информации для передачи или хранения сообщения над его информационной энтропией.

Энтропия

Под информационной энтропией понимается мера неопределенности или случайности в информации. Она показывает, сколько информации содержится в сообщении. Чем выше энтропия, тем больше информации необходимо для её описания.

Формула Шеннона для расчета энтропии
Шум

Шумом называют всякие нежелательные изменения в сигнале, которые могут привести к искажению информации во время передачи. Шум может возникать из-за различных факторов, такие как электромагнитные помехи, физические препятствия и т.д.

Канал связи

Каналом связи называется средство, через которое передается информация, например, радиоволны, оптоволокно, электрические сигналы. Каналы могут иметь различные характеристики, такие как пропускная способность и уровень шума.

Передатчик

Передатчиком называется устройство или система, которая отправляет информцию. Это может быть, например, компьютер, телефон или радиостанция

Приемник

Приемником называется устройство или система, которая получает информцию. Это может быть, например, компьютер, телефон или радиостанция

Алгоритм Хаффмана

Алгоритм Хаффмана — жадный алгоритм оптимального префиксного кодирования алфавита с минимальной избыточностью.

Условие Фано

Звучит оно примерно так: "Никакое кодовое слово не может быть началом другого кодового слова".

Сжатие данных

Сжатием называется процесс уменьшения объема информации, который может быть достигнут с помощью различных алгоритмов.

Шум

Шумом называется любое нежелательное изменение сигнала, которое может исказить передаваемую информацию.

Пропускная способность канала

Пропускной способностью канала называется максимальное количество информации, которое может быть передано через канал за единицу времени. Обычно измеряется в битах в секунду (bps).
